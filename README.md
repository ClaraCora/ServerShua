# Shua - VPS 流量消耗与定时下载助手

这是一个轻量级的 Linux VPS 流量消耗脚本。通过定时下载指定文件来消耗服务器流量，同时将下载数据直接丢弃到 `/dev/null`，**实现零硬盘占用、零硬盘磨损**。

这个工具非常适合用来跑满那些标称 50M 带宽、每月有 512GB 等大额双向流量配额，但平时处于闲置状态的机器，确保资源不被浪费。

## ✨ 核心特性

- **零磁盘损耗**：数据直接进入内存并销毁，完全不写盘。
- **精准统计**：调用 `curl` 内部接口，精准统计真实消耗的流量，自动换算为 GB 并保留三位小数。
- **灵活的时间窗口**：支持设置多个同天内的工作时间段（如 `08:00-12:00,14:00-18:00`），其余时间自动休眠。
- **全局面板控制**：一键呼出交互式面板，随时修改下载链接、时间间隔，配置即时生效。
- **Systemd 守护**：后台静默运行，开机自启，防崩溃断联。

## 🚀 安装指南

请使用 `root` 用户登录你的 VPS，并执行以下命令：

```bash
# 1. 克隆项目到本地
git clone [https://github.com/你的用户名/shua.git](https://github.com/你的用户名/shua.git)
cd shua

# 2. 运行一键安装脚本
bash install.sh
